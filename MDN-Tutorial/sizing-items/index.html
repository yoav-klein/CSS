max-width

max-width is useful when working with images. setting a max-width: 100% on an image allows the image to scale down when there's not enough space in the enclosing container, while not causing it to stretch when the container is bigger than then intrinsic value of the image


in the first example, we use width: 100% so the image is streched, causing pixalation

in this example, we use max-width:100% so that the image doesn't strech

Viewport

Viewport is the visible area of your page in the browser. The vh (viewport height) and vw (viewport width) are units relative to the size of the viewport.
