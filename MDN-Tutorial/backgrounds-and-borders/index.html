Repeat mechanisms

When the image is smaller than the box, we set the background-repeat to determine what to do

background-repeat: no-repeat
background-repeat: repeat-x
background-repeat: repeat-y

Background size

When the image is larger than the box, we set the background-size to determine what to do

cover

background-size: cover makes that the image will cover the box while retaining the ratio of the image. this causes that the image will be cropped unless the ratio of the box is exactly the same

contain

background-size: contain makes the image the size that it will fit wholy inside the box. This may create gaps if the ratio is not equal

Background position

You use the background-position to set where in the box the image will appear

this is the default
background-position: 20px 20px
background-position: 50% 50%
background-position: 20px top
