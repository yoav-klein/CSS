Standard model vs Alternate model

Alternate model (box-sizing: border-box) sets the height and width on content+padding+border
Standard model (box-sizing: content-box) sets the height and width on content only

inline-block

The inline-block display is a middle ground between inline and block. It doesn't break into a new line, but respects height and width, and padding, borders and margins are respected top-bottom wise.
