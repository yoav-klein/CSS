# Heights and Overflows
---

In this series of examples I investigate the behavior of CSS when it comes to heights of elements and setting `overflow: auto` on them.
There are some quirky stuff when it comes to flexbox and layouts.

You can expand your knowledge about this by reading about Block Formatting Contexts, Formatting Contexts, stuff like that:
https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Display/Formatting_contexts
https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Display/Block_formatting_context
